Pour faire fonctionner la lecture du fichier de données du professeur, 
il faut ajouter dans le dossier "log430Team07" le fichier "detector.txt".

Une fois l'API lancer, vous pouvez aller sur "localhost:3000/mqtt" pour recevoir les données du mqtt.
"localhost:3000/mqtt/dataFile" pour recevoir les données du fichier "detector.txt".
